1. в пустом проекте npm init -y
2. поставить нужные зависимости (npm install cross-env express lowdb nodemon)
3. прописать в скриптах ("dev": "cross-env NODE_ENV=development nodemon server.js")
4. создать файлик server.js, прописать нужные команды, прописать нужные команды для запуска сервера.
Запустить сервер с помощью (npm run dev), но пока у нас не реализованы нужные нам роуты.
5. Создадим папочку routes, где cоздадим файлик routes.js, где создадим объект.
